# Plantilla Profesional Frontend (HTML/CSS/JS)

Proyecto base listo para trabajar con HTML, CSS y JavaScript vanilla, usando Vite para desarrollo y producción.

## Scripts disponibles

- `npm install` — instala dependencias
- `npm run dev` — servidor local con recarga en vivo
- `npm run build` — genera la versión optimizada en `/dist`
- `npm run preview` — previsualiza la build

## Estructura
frontend-template/
│
├─ src/

│   ├─ css/
│   │   ├─ main.css
│   │   └─ components/
│   │       └─ button.css
│   │
│   ├─ js/
│   │   ├─ main.js
│   │   └─ modules/
│   │       ├─ dom.js
│   │       └─ utils.js
│   │
│   ├─ assets/
│   │   ├─ img/
│   │   ├─ icons/
│   │   └─ fonts/
│   │
│   └─ styles/

├─ .gitignore

## Características
- Modularización de JS
- Estructura profesional escalable
- Vite para entorno moderno

- Listo para GitHub Pages o cualquier host
